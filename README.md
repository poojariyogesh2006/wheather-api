# wheather-api
🌤️ Advanced Weather Dashboard

A modern Python-based Weather Dashboard built with Tkinter, OpenWeatherMap API, and Matplotlib. The application provides real-time weather information, 5-day forecasts, weather analytics, air quality monitoring, and weather alerts through an interactive graphical user interface.

🚀 Features
🌍 Real-time weather data for any city
📅 5-Day weather forecast
🌡️ Temperature display in Celsius and Fahrenheit
💨 Wind speed and direction monitoring
💧 Humidity and pressure information
☁️ Cloud coverage details
👁️ Visibility tracking
🌬️ Air Quality Index (AQI) monitoring
📊 Weather graphs and trend visualization
⚠️ Smart weather alerts and warnings
🎨 Modern and user-friendly GUI
📍 Location-based weather search
🔄 Live weather updates
🛠️ Technologies Used
Python
Tkinter
OpenWeatherMap API
